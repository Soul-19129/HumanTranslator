## 📖 حول المشروع | About the Project

**HumanTranslator** هو مشروع مفتوح المصدر يهدف إلى توفير حلول ترجمة فورية ومتقدمة لمساعدة الفئات الاجتماعية المختلفة، بما في ذلك:

**HumanTranslator** is an open-source project that aims to provide advanced real-time translation solutions to help various social groups, including:

- 👨‍🎓 **الطلاب** | Students
- 🏠 **اللاجئين** | Refugees
- 🔇 **ضعاف السمع** | Hearing-impaired individuals
- 🌐 **المجتمعات متعددة اللغات** | Multilingual communities

## 🚀 كيفية الاستخدام | How to Use

### 📥 التثبيت | Installation

```bash
# استنساخ المستودع | Clone the repository
git clone https://github.com/Soul-19129/HumanTranslator.git

# الانتقال إلى المجلد | Navigate to directory
cd HumanTranslator

# تثبيت الحزم المطلوبة | Install dependencies
npm install
# أو | or
pip install -r requirements.txt

# تشغيل التطبيق | Run the application
npm start
# أو | or
python app.py
```

### 🌐 الاستخدام عبر الويب | Web Usage

1. 🌍 **افتح المتصفح** | Open browser
2. 📝 **أدخل النص المراد ترجمته** | Enter text to translate
3. 🔄 **اختر اللغات** | Select languages
4. ✅ **احصل على الترجمة** | Get translation

### 🎤 الاستخدام الصوتي | Voice Usage

1. 🎙️ **اضغط على زر التسجيل** | Press record button
2. 🗣️ **تحدث بوضوح** | Speak clearly
3. 🔄 **اختر لغة الترجمة** | Choose translation language
4. 🔊 **استمع للترجمة** | Listen to translation
